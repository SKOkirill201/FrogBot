"Как добавить бота ко мне в чат?"

Для этого нам нужно перейти в меню добавления участников, затем пишем в поиске "@frog2bot", затем добавляем с оффициальной иконкой.

Если вы по каким-либо причинам не можете пользоватся ботом в чате, им можно пользоватся через личную переписку.
